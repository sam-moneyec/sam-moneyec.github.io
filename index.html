Producto Cartesiano ($A \times B$)

Slide 2

Definición Formal

[cite_start]

Dados dos conjuntos A y B, el producto cartesiano es el conjunto de todos los pares ordenados $(a,b)$ tal que $a \in A$ y $b \in B$[cite: 7].

$$A \times B = \{(a,b) | a \in A \land b \in B\}$$ $$|A \times B| = |A| * |B|$$

Definición (Mapeo)

[cite_start]

Una función $f: A \rightarrow B$ es una asignación donde a cada elemento $x$ de $A$ se le asigna un único elemento $y = f(x)$ de $B$[cite: 19].

Ejemplos y Validación

Slide 8

¿Es Función?

[cite_start]

Analicemos la relación $R=\{(2,5), (4,1), (4,3), (6,5)\}$ con $A=\{2,4,6\}$ y $B=\{1,3,5\}$[cite: 120].

No es Función

El elemento 4 tiene dos salidas (1 y 3). En una función, cada entrada debe tener una ÚNICA salida.

Sí es Función

Si tuviéramos $\{(2,5), (4,1), (6,5)\}$. No importa que el 5 se repita en la llegada, lo importante es que la salida sea única.

Clasificación de Funciones

Slides 9-10

Inyectiva

[cite_start]

Cada elemento de B tiene a lo sumo una anti-imagen en A (Uno a Uno)[cite: 136].

Si $f(x_1)=f(x_2) \implies x_1=x_2$

Sobreyectiva

El Rango es igual al Codominio. [cite_start]Todo elemento de B recibe flecha[cite: 137].

$f(A) = B$

Biyectiva

[cite_start]

Es inyectiva y sobreyectiva simultáneamente[cite: 138].

$|A| = |B|$
Validación Inyectividad (MATLAB)
n = 1:10; % Dominio
y = 2*n + 1; % Función f(n)
if length(unique(y)) == length(y)
    disp('Es Inyectiva: No hay colisiones');
end

Función Inversa ($f^{-1}$)

Slide 11
[cite_start]Requisito: Para que exista inversa, la función debe ser Biyectiva[cite: 164].

La inversa invierte el "flujo" de la máquina. Lo que antes era salida, ahora es entrada.

A $\xrightarrow{f}$ B
A $\xleftarrow{f^{-1}}$ B
