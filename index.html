Producto Cartesiano

Teoría de Conjuntos

Dados dos conjuntos A y B, el producto cartesiano $A \times B$ es el conjunto de todos los pares ordenados (a,b).

[cite_start]

$$A \times B = \{(a,b) | a \in A \land b \in B\}$$ [cite: 7, 9]

Calculadora Interactiva

MATLAB Code

[X, Y] = ndgrid(A, B);
disp([X(:), Y(:)]);

Definición de Funciones

[cite_start]

Una función es una asignación donde a cada elemento $x$ de $A$ se le asigna un único elemento $y = f(x)$ de $B$[cite: 19].
