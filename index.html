1. La Máquina Funcional

Una función puede verse como una máquina: entra un insumo $x$ (Input), se transforma, y sale un producto $f(x)$ (Output)[cite: 18, 45, 50].

MATLAB Code 
% Simulación de Máquina
x = input('Ingrese valor: ');
f = x^2; % Ejemplo cuadrático
fprintf('Output: %d', f);

2. Producto Cartesiano

El conjunto de todos los pares ordenados $(a,b)$ donde $a \in A$ y $b \in B$[cite: 7, 9].

3. Composición $(f \circ g)(x)$

La composición aplica una función dentro de otra: $(f \circ g)(x) = f(g(x))$.

Ejemplo: Sea $f(x) = x^2$ y $g(x) = x + 1$.

Resultado aparecerá aquí
MATLAB Code 
% Composición f(g(x))
g = @(x) x + 1;
f = @(x) x.^2;
resultado = f(g(x));
