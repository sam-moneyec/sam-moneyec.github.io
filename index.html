Producto Cartesiano Clasificación Funciones Discretas

1. Producto Cartesiano ($A \times B$)

Es el conjunto de todos los pares ordenados $(a,b)$ tal que $a \in A$ y $b \in B$[cite: 7, 9].

Código MATLAB:

A = [1, 2, 3]; B = {'u', 'v'};
[X, Y] = ndgrid(A, B);
producto = [X(:), Y(:)];
disp(producto);

2. Clasificación de Funciones

Una función es un "mapeo" donde a cada elemento de $A$ le corresponde un único elemento de $B$[cite: 18, 19].

Inyectiva: Cada elemento de B tiene a lo sumo una anti-imagen[cite: 136].
Sobreyectiva: El rango es igual al codominio[cite: 137, 160].

Validación en MATLAB (Inyectividad):

% Si f(n) = 2n + 1 es inyectiva
n = 1:10; y = 2*n + 1;
if length(unique(y)) == length(y)
    disp('Es Inyectiva');
end

3. Funciones Discretas

Dominio en los enteros ($\mathbb{Z}$) y codominio en los reales ($\mathbb{R}$)[cite: 222, 223].

Código MATLAB:

n = 5;
resultado = factorial(n);
disp(resultado);
