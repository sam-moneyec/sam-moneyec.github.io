TEORÍA DE CONJUNTOS

Capítulo 1

¿QUÉ ES?

El Producto Cartesiano es una operación matemática que toma dos conjuntos y los mezcla para crear uno nuevo. Su idea central es formar pares ordenados, donde el primer elemento proviene del primer conjunto y el segundo del segundo conjunto.


Cada elemento del primer conjunto se combina sin excepción con todos los del segundo. El resultado se escribe como:

$$A \times B$$

Y refleja todas las combinaciones posibles. Eso sí, el orden manda: cambiar los elementos cambia completamente el par.

$$A \times B = \{(a,b) \mid a \in A \land b \in B\}$$

Cardinalidad: $|A \times B| = |A| * |B|$

LA MÁQUINA

Una función $f: A \rightarrow B$ es una asignación donde a cada elemento $x$ de $A$ se le asigna un único elemento $y = f(x)$ de $B$.

VALIDACIÓN

Slide 8

¿ES FUNCIÓN?

Analicemos la relación $R=\{(2,5), (4,1), (4,3), (6,5)\}$ con $A=\{2,4,6\}$ y $B=\{1,3,5\}$.

CASO ERROR

El elemento 4 tiene dos salidas (1 y 3). ¡Esto rompe la regla de unicidad!

CASO VÁLIDO

Si eliminamos $(4,3)$, cada elemento tendría una sola flecha. No importa si el 5 recibe dos flechas.

CLASIFICACIÓN

Slides 9-10

Inyectiva

Cada elemento de B tiene a lo sumo una anti-imagen. (1 a 1).

Si $f(x_1)=f(x_2) \implies x_1=x_2$

Sobreyectiva

El Rango es igual al Codominio. No sobran elementos en B.

$f(A) = B$

Biyectiva

Es inyectiva y sobreyectiva simultáneamente. Perfecta.

$|A| = |B|$
Validación Inyectividad
n = 1:10; 
y = 2*n + 1; 
if length(unique(y)) == length(y)
    disp('Es Inyectiva: No hay colisiones');
end

COMPOSICIÓN

La composición se denota $(f \circ g)(x) = f(g(x))$. Si $f$ y $g$ son biyectivas, su composición también lo es.
