TEORÍA DE CONJUNTOS

NIVEL 1

1. DEFINICIÓN TÉCNICA

En matemáticas, el producto cartesiano es una operación crítica que fusiona dos conjuntos para generar un nuevo espacio dimensional.

La Fórmula Maestra

Dados los conjuntos A y B, el producto cartesiano (A x B) es el conjunto de todos los pares ordenados (a, b) posibles.

A x B = { (a, b) | a ∈ A y b ∈ B }

¡Alerta! El orden es absoluto. El par (1, 2) NO es lo mismo que el par (2, 1). Es la diferencia entre coordenadas distintas en el mapa.

VISUALIZACIÓN Y CÓMPUTO

Diagrama de Flechas

Código Generador

% Definimos los conjuntos
A = [1, 2, 3];
B = [4, 5];

% Generación eficiente (Ingeniería)
[GridA, GridB] = ndgrid(A, B);

% Aplanado y Unión
Pares = [GridA(:), GridB(:)];

% Resultado
disp('--- A x B ---');
disp(Pares);
% Resultado esperado:
% 1  4
% 2  4
% 3  4
% 1  5 ...

APLICACIONES REALES

Esto no es teoría. Es el motor de la tecnología que usas:

1. Resolución de Pantalla

Tu monitor Full HD es un Producto Cartesiano gigante de 1920 columnas x 1080 filas. Cada píxel es un par ordenado (x, y).

2. Bases de Datos (SQL)

Un CROSS JOIN en SQL es un producto cartesiano. Si se usa mal, combina cada fila de una tabla con todas las de otra.

3. Criptografía

Un ataque de fuerza bruta recorre el producto cartesiano de todos los caracteres posibles para romper una contraseña.

CLASIFICACIÓN DE FUNCIONES

NIVEL 3

INJECTIVA · SOBRAYECTIVA · BIYECTIVA

Una función f: A → B puede clasificarse según cómo “cubre” el codominio y si repite salidas:

Injectiva

No repite valores de salida: si x₁≠x₂ entonces f(x₁)≠f(x₂).

Sobreyectiva

Cubre todo el codominio: para cada y∈B existe x∈A con f(x)=y.

Biyectiva

Es injectiva y sobreyectiva a la vez. Tiene inversa.

FUNCIÓN INVERSA

NIVEL 4

REGLA CLAVE

Una función tiene inversa f⁻¹ si y solo si es biyectiva. La inversa “invierte las flechas”: si f(x)=y entonces f⁻¹(y)=x.

f^{-1}: B → A

FUNCIÓN COMPUESTA

NIVEL 5

CONCEPTO

La composición encadena dos funciones: primero aplicas f y luego g. Si f: A→B y g: B→C, entonces la compuesta es:

(g \circ f)(x) = g(f(x))

En palabras gamer: input → f → resultado intermedio → g → output final.
